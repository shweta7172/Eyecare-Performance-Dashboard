# 📊 Eyecare Performance Dashboard

A Power BI dashboard built to help eyecare retail leadership monitor 
store performance across multiple locations.

## 🔗 Live Dashboard
[View Published Dashboard](https://app.powerbi.com/links/nSo8nweoGh?ctid=ff28a236-881c-4b29-9b77-71405b66aca2&pbi_source=linkShare)

## 📌 Business Problem
An eyecare retail chain needed a unified analytics solution to track:
- Revenue metrics
- Appointments & wait times
- Store performance

## 🎯 Project Objectives
Build an interactive Power BI dashboard to help leadership:
- Monitor core KPIs
- Identify bottlenecks in operations
- Support data-backed decision making

## 📈 Key Metrics
| Metric | Value |
|---|---|
| Total Revenue | €1.48M |
| Total Appointments | 21K |
| Conversion Rate | 60.76% |
| Average NPS | 66 |

## 🏪 Stores Covered
- Store 1 – Munich
- Store 2 – Cologne
- Store 3 – Frankfurt
- Store 4 – Berlin
- Store 5 – Hamburg

## 🔍 Key Insights
- **Munich** performs strongest: high attendance, low wait times, high NPS
- **Hamburg** shows highest no-show rate and lowest NPS — operational bottleneck
- Wait time and NPS are directly correlated across all stores
- Purchase mix: 54% Frames, 35% Lenses, 11% Contacts

## 🛠️ Technologies Used
- Power BI Desktop
- DAX Calculations
- Data Modeling
- Excel (data source)
